Created with CodeSandbox

📸 About the Project

The Potluck Guest List app lets you keep track of who's coming to your next potluck. Users can add guests to a dynamic list in real time — no page reloads, no fuss.
This project was built to practice core front-end skills including DOM manipulation, event handling, and dynamic UI updates using vanilla JavaScript.


✨ Features

Add guests to a live, dynamic guest list
Instant UI updates without page refresh
Clean, minimal interface


🛠️ Built With

HTML — Page structure and markup
CSS — Styling and layout
JavaScript — DOM manipulation and interactivity
